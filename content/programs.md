I prefer to focus on minimalism and productivity. Also, I try to use free and open source programs as much as possible, and encourage their use. I'm a fan of the [GNU Project](https://www.gnu.org/home.en.html). Because of flexibility and speed, I prefer terminal over GUI.

#### Operating System
I use [Arch Linux](https://www.archlinux.org/). It's a lightweight and flexible Linux distribution. It provides packages in a rolling release model. So, I wouldn't recommend it to you if you are a beginner linux user.

#### Terminal emulator
I'd pick [Alacritty](https://github.com/alacritty/alacritty) over any other terminal. It's fast, it's customizable, and it supports unicode fonts. Did I mention that it's fast?

#### Shell
I use [zsh](https://www.zsh.org/) with [oh-my-zsh](https://github.com/ohmyzsh/ohmyzsh) configuration manager. There are countless themes and plugins available for zsh. I've used bash in the past, but once you start using zsh, believe me, you'll never go back. 

#### Window Manager / Desktop Environment
I use [i3](https://i3wm.org/) window manager. It's a tiling wm that is highly customizable and well documented. 

Speaking of Desktop Environment, I used to use [XFCE](https://www.xfce.org/) and [KDE](https://kde.org/) before switching over to i3. Both are equally good DEs. Both of them are highly customizable and resource light. If you're on an old machine, I'd recommend XFCE as it is lighter than KDE.

#### Text editor
[Vim](https://www.vim.org/), the greatest text editor ever made. It's not just a text editor, it's a lifestyle to me. 

#### Web browser
I'm concerned about my online privacy, so I use [Brave](https://brave.com/). It's a privacy focused browser based on chromium engine. It comes with prebuilt ad blocker. I occasionally use [Firefox](https://www.mozilla.org/en-US/firefox/new/).

#### File manager
I use [nnn](https://github.com/jarun/nnn). It's a terminal based file manager that is lightweight, customizable, and comes with many useful functionalities like bulk renaming, regex search, and many more.

#### Media player
I use [mpv](https://mpv.io/). It can play music, videos, and yes, it can even stream YouTube videos.

#### RSS reader
I don't use social media a lot. So, for keeping up with the news, I use an RSS reader, specifically [Newsboat](https://newsboat.org/). Read more about RSS [here](https://en.wikipedia.org/wiki/RSS).

#### Torrent client
I use [qBittorrent](https://www.qbittorrent.org/). It's free and open source, has no ads, and works perfectly.

#### PDF viewer
[Zathura](https://pwmt.org/projects/zathura/). It supports PDF, postscript, DJVU, and many other file formats through plugins. It has keyboard bindings, so no mouse is needed. Also, it's highly customizable.

#### Image viewer
I use [feh](https://feh.finalrewind.org/). It is very light and powerful. It can also be used to set desktop wallpaper. It is just for viewing images. For editing images, I use [GIMP](https://www.gimp.org/).
