Programs I Use

Home

I've been hooked to terminal based applications for quite a while now. They are fast, and a lot flexible compared to GUI programs. That's not to say I completely avoid GUI applications. Both have their pros and cons. I think identifying the pros of both sides and developing the ability to harness them is best choice for the user.

Operating System

I use Arch Linux. It's a lightweight and flexible Linux distribution. It provides packages in a rolling release model. So, I wouldn't recommend it to you if you are a beginner linux user.

Terminal emulator

Alacritty is one of the fastest out there. It's also highly customizable, and it supports unicode fonts. I am using Konsole these days because it integrates well with KDE.

Shell

I use zsh with oh-my-zsh configuration manager. There are countless themes and plugins available for zsh. I've used bash in the past, but once you start using zsh, believe me, you'll never go back.

Window Manager / Desktop Environment

Some days I use i3 window manager, other days, I use KDE. i3 is highly customizable and very well documented. It's a great choice if you don't want to deal with mouse that much and like getting things done from keyboard.

Speaking of KDE, it's a customizable and lightweight Desktop Environment. I used to use XFCE but switched to KDE because of its aesthetics and ecosystem.

Text editor

Vim, the greatest text editor ever made. Believe me, you'll never regret learning vim.

Emacs is my IDE these days. Even this website was made using emacs!

Web browser

I'm concerned about my online privacy, so I use Brave. It's a privacy focused browser based on chromium engine. It comes with prebuilt ad blocker. I occasionally use Firefox.

File manager

I use nnn. It's a terminal based file manager that comes with many useful functionalities like bulk renaming, regex search, etc.

Nothing beats Dolphin amongst the GUI file managers. Give it a try!

Media player

I use mpv. It can play music, videos, and yes, it can even stream YouTube videos. It's faster than VLC.

RSS reader

I don't use social media a lot. For keeping up with the news, I use an RSS reader, specifically Newsboat. Read more about RSS here.

Torrent client

I use qBittorrent. It's free and open source, has no ads, and works perfectly.

PDF viewer

Zathura. It supports PDF, postscript, DJVU, and many other file formats through plugins. It has keyboard bindings, so no mouse is needed. Also, it's highly customizable.

Okular is undefeated amongst open source PDF viewers. It's much more than a PDF viewer.

Image viewer

I use feh for casual image viewing. It is light and powerful. It is just for viewing images. For editing images, I use GIMP.
